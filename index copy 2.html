Kat Klimek

I’m a creative student from Roehampton London University exploring dark, cinematic storytelling through web design.

This website is a small portfolio-style project built with HTML, CSS, and JavaScript. I’m interested in film-inspired aesthetics (especially Darren Aronofsky’s mood and intensity), typography.

On this site you’ll find a text-focused page, a gallery page, and a mixed layout page. Everything is designed to be responsive and accessible.
